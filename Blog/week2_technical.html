JavaScript: Why so popular?

For this unit of DBC, we've been asked to write a post on why JavaScript is so popular in web development. From my limited experience I know that--if HTML is the structure of a web page and CSS is the style of a web page then JavaScript is the action of a webpage...or how elements behave.

So JavaScript is the scripting language of the web. It gives us all the cool web bells and whistles we are so used to like slideshows, autocomplete, and pop-ups. But why eles is JavaScript so popular?

Here's my take:

  1. Easy to learn
  2. Can be used on the client as well as server side
  3. It's fast and responsive. It works well with the server which means more efficiency for the user

JavaScript is becoming a high-demand skill. It seems that getting up on your JavaScript game is super beneficial to becoming an in demand developer.
